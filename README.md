# Gen-Z Online Shopping Behavior — Survey Analysis

## About This Project
I was curious about how people my age actually shop online —
not just what platforms they use, but why they stay loyal,
what makes them switch, and how much social media really
influences what they buy.

So I designed a survey, collected responses from 54 Gen-Z
individuals across India, and built an interactive dashboard
to explore the patterns in our shopping behavior.

## How I Collected the Data
I created a structured questionnaire using Google Forms covering
22 questions — from basic demographics to spending habits,
discount preferences, platform loyalty, and social media influence.

- Responses : 54 Gen-Z respondents
- Survey Period : January 2026
- Region : Mostly Mumbai + other Indian cities

Raw data is not uploaded here to protect respondent privacy.

## What I Built

An interactive Shiny dashboard built in R that lets you explore
the survey results visually — filter by gender, city, platform,
and more to see how different segments of Gen-Z shop differently.

## What I Looked At

- Which platforms Gen-Z uses most and why they stick to them
- How much they spend yearly, monthly, and per order
- What kind of discounts actually drive purchases
- How much online reviews and social media affect buying decisions
- What would make someone switch to a different platform

## Tech Stack

- **Google Forms** — survey design and data collection
- **R / RStudio** — data cleaning, analysis, and app development
- **Shiny + shinydashboard** — interactive web dashboard
- **ggplot2 + plotly** — data visualizations
- **dplyr + stringr + scales** — data wrangling

## How to Run

1. Clone this repo
2. Open the `.R` script in RStudio
3. Install dependencies if needed:

install.packages(c("shiny", "shinydashboard", "readr",
                   "plotly", "ggplot2", "dplyr", 
                   "stringr", "scales"))

4. Click **Run App** in RStudio
